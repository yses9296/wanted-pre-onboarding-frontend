# 원티드 프리온보딩 프론트엔드 - 선발 과제

## `npm start`

$ `npm install` </br>
$ `npm start`

<p> 패키지를 설치후 코드를 실행하시면 localhost:8000 포트에 서버가 실행됩니다. </p>
<p> 메인 화면은 회원가입(SignUp) 페이지로 접근합니다. </p>
